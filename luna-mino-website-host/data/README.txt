===================================
   Website Host - Data Folder
===================================

This folder contains your website files.

HOW TO USE:
-----------
1. Place your HTML, CSS, JavaScript, and PHP files in this folder
2. The index.html or index.php will be your homepage
3. You can create subdirectories for organization
4. All PHP 8.2 features are supported

EXAMPLES:
---------
- Static Website: Just add your HTML/CSS/JS files
- PHP Website: Create PHP files and they'll be executed
- WordPress: Extract WordPress files here
- Any PHP application compatible with Apache

FILE STRUCTURE:
---------------
data/
  ├── index.html (or index.php) - Your homepage
  ├── css/
  ├── js/
  ├── images/
  └── ... (your other files)

NOTES:
------
- Files are persistent across container restarts
- Maximum upload size depends on PHP configuration
- Default PHP 8.2 with Apache
- Access via Umbrel dashboard or configured port

Need help? Check the app description in the Umbrel App Store.
